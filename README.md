# AI-job-recommender

一个基于 AI 的简历智能推荐系统，帮助用户根据简历内容匹配最适合的岗位。

## 项目结构
- `前端/`：用户上传简历、查看推荐结果的界面（HTML + JS）
- `后端/`：处理简历解析、岗位匹配逻辑的 API（Spring Boot + MySQL）

## 技术栈
- 前端：HTML / CSS / JavaScript
- 后端：Spring Boot / MySQL
- AI 集成：Dify 工作流
- 部署：GitHub Pages

## 如何使用
1. 克隆仓库：`git clone https://github.com/Layayaa/ai-job-recommender.git`
2. 启动后端服务（Spring Boot）
3. 打开前端 `index.html` 即可使用

## 项目亮点
- 实现简历自动解析与岗位智能匹配
- 支持实时加载与错误重试机制
- 代码结构清晰，前后端分离，便于扩展
